# Source Inventory

## Paper Sources

- `dronet/RAL18_Loquercio.pdf` - Original DroNet paper.
- `dronet/paper_text.txt` - Extracted text from the paper, generated by `read_pdf.py`.
- `read_pdf.py` - Helper for extracting paper text.

## Upstream Repository

- `dronet/repo/README.md` - Upstream project description, setup, training, evaluation, and citation.
- `dronet/repo/cnn_models.py` - Keras ResNet-8 model definition.
- `dronet/repo/cnn.py`, `dronet/repo/evaluation.py`, `dronet/repo/utils.py`, `dronet/repo/common_flags.py` - Training/evaluation and data pipeline.
- `dronet/repo/img_utils.py` - Original image load, resize, and central-bottom crop behavior.
- `dronet/repo/model/model_struct.json`, `dronet/repo/model/model_weights.h5` - Upstream model structure and weights used locally.
- `dronet/repo/images/architecture.png`, `dronet/repo/images/dataset.png` - Paper/repo diagrams.

## ROS/Bebop Control

- `dronet/repo/drone_control/dronet/README.md` - Instructions for ROS/Bebop operation.
- `dronet/repo/drone_control/dronet/dronet_perception/` - ROS perception package wrapping the Keras model.
- `dronet/repo/drone_control/dronet/dronet_control/` - ROS control package converting network outputs to Bebop velocity commands.

## Local Adaptation

- `dronet/dronet_model.py` - PyTorch model port and preprocessing.
- `dronet/run_dronet.py` - Batch video inference and annotated-video generation.
- `dronet/selftest.py` - Smoke test for model loading, preprocessing shape, and sample predictions.
- `dronet/inspect_h5.py`, `dronet/probe.py` - Model/weight inspection helpers.
- `dronet/make_examples.py` - Creates 20 example image tiles and a contact sheet from processed video frames.

## Generated Results

- `dronet_results/summary.json` - Per-clip aggregate statistics.
- `dronet_results/all_frames.csv` - Combined per-frame predictions.
- `dronet_results/*.csv` - Per-video per-frame predictions.
- `dronet_results/*_annotated.mp4` - Annotated visual outputs.
- `dronet_results/examples/*.png` and `examples_contact_sheet.png` - Model input crops plus predictions.
- `dronet_results/examples_100_20fps/*.png` and `examples_100_20fps_contact_sheet.png` - 100 continuous 20 FPS samples across the concatenated readable clips.
- `dronet_results/sensecv_20_21_10fps/*.png` and `sensecv_20_21_10fps_contact_sheet.png` - DroNet crops/predictions for SenseCV clips 20 and 21 sampled at 10 FPS.
- `dronet_results/pilotguru_10fps_match/summary.csv`, `summary.json`, `samples.csv`, and `sampled_frames/` - 10 FPS sampled-frame inference pass over all readable PilotGuru export videos, including folder-name expectation matching summarized in [[pilotguru-10fps-folder-check]].
